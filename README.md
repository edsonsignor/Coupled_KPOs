# Coupled Kerr Parametric Oscillators

Numerical simulations of coupled Kerr parametric oscillators with applications to nonlinear dynamics and quantum computing.


## Repository structure

src/        Core source code
scripts/    Reproducible simulation scripts
notebooks/  Exploratory Jupyter notebooks
figures/    Generated plots (small outputs only)

## Requirements

- Julia ≥ 1.9
- Python ≥ 3.9 (optional, for analysis and plotting)
- JupyterLab (optional)

## Status

Active development. APIs and structure may change.
